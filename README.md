# Tech Sales Data Cleaning & Analysis

This project simulates a take-home assignment for a Junior Data Analyst role at a SaaS company.

## Objective
Clean messy CRM export data and generate insights about revenue, regional performance, and sales representatives.

Pandas  
Google colab
